# Success369 | Clarity. Congruence. Catalysis.

Success369 is a premium, cinematic web experience designed to facilitate sustainable success through clarity and alignment.

## Core Pillars

- **Foundation**: Aligning who you are with what you do.
- **Programs**: Distinct journeys following a shared philosophy.
- **Sustainable Success**: Progress that feels steady, coherent, and deeply fulfilling.

## Key Features

- **Cinematic Hero Slider**: Interactive, video-background experiences.
- **Premium UI/UX**: Glassmorphism, high-contrast protection, and standardized precision spacing.
- **Advanced Animations**: Powered by Framer Motion for smooth, high-end transitions.
- **Responsive Branding**: Integrated logo, custom tagline, and brand-consistent color systems.

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Typography**: Nunito (Google Fonts)

## Local Development

1. **Install Dependencies**: `npm install`
2. **Start Dev Server**: `npm run dev`
3. **Build for Production**: `npm run build`
